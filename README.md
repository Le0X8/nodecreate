<h1 align="center">nodecreate</h1>
<p align="center">create a new Node.js project with everything you need</p>


## Usage

```bash
npm create leolesinski@latest <path>
```

## Features

- npm configuration
- Package name availability check
- README template
- License template
- Directory structure
